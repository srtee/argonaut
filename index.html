Welcome to Argonaut!

A browser-based bibliography manager that stores your papers as JSON.

Load Your Papers

Import your collection from a JSON file, URL, browser storage, or GitHub Gist using the Load JSON Collection section.

Add Papers by DOI

Quickly add papers by entering a DOI or URL in the Add A Paper By DOI section. We'll fetch the BibTeX and abstract automatically.

Save & Export

Save your collection to a file, browser storage, or GitHub Gist. Export as BibTeX for use in LaTeX or other citation managers.

Organize Your Papers

Add tags to filter and group your papers. Use the comment box on each paper to add your notes and thoughts.

You're Ready!

Click "Get Started" to begin using Argonaut. You can view this guide again from the help link in the footer.
